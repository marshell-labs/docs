title: Dashboard & syncing
description: Sign in to sync your checks to the chifu dashboard, and manage your API keys.
---

Every `chifu check` requires an account. When signed in, results sync to [marshell.dev/dashboard](https://marshell.dev/dashboard) with history per repo.

## Sign in

**Browser pairing (recommended)**

```bash
chifu login
```

Opens a pairing URL in your browser. No copy-pasting secrets. Key saved to `~/.config/chifu/config.json` (mode `600`).

**Manual key**

```bash
chifu login chf_xxxxxxxx
```

Or set `CHIFU_API_KEY`. Keys start with `chf_`.

The [wizard](/quickstart) can sign you in during install (`--api-key` or browser flow).

## API keys

Create and revoke keys in the dashboard under **Settings**. Use keys for:

- Local CLI (`chifu login` or `CHIFU_API_KEY`)
- CI (`CHIFU_API_KEY` secret)
- Never commit keys to git

## What syncs

Each authenticated `chifu check` records:

- Vulnerable package count and severity rollup
- Repo identity (from `git remote` when available)
- Timestamp for history and charts

The dashboard does **not** re-run CVE matching — it displays what the CLI reported.

## Usage limits

Plans meter checks per month. See [Plans](/plans).

## Log out locally

```bash
chifu logout
```

Clears the local key. Dashboard account unchanged.

The installer detects the coding agents on your machine and writes the
`chifu-dep-guard` skill into each one, in that agent's native format. One
command, every agent.

## Supported agents

| Agent | Format | Location |
|---|---|---|
| Claude Code | Skill | `~/.claude/skills/chifu-dep-guard/SKILL.md` |
| Cursor | Project rule | `~/.cursor/rules/chifu-dep-guard.mdc` |
| Windsurf | Memory rule | `~/.codeium/windsurf/memories/chifu-dep-guard.md` |
| Codex | `AGENTS.md` block | `~/.codex/AGENTS.md` |
| OpenCode | `AGENTS.md` block | `~/.config/opencode/AGENTS.md` |
| Gemini CLI | `GEMINI.md` block | `~/.gemini/GEMINI.md` |
| Cline | Rule file | `~/.clinerules/chifu-dep-guard.md` |

<Note>
  Writes to `AGENTS.md` and `GEMINI.md` are idempotent. chifu inserts a
  delimited block and replaces just that block on future updates, leaving the
  rest of your file untouched.
</Note>

## Install for specific agents

By default the installer offers every agent it detects. To target specific
ones non-interactively:

```bash
# install for everything detected, no prompts
npx @marshell/chifu-wizard@latest --yes

# install for one or more named agents
npx @marshell/chifu-wizard@latest --target claude --target cursor
```

Valid targets are `claude`, `cursor`, `windsurf`, `codex`, `opencode`,
`gemini`, and `cline`.

Other useful flags:

| Flag | What it does |
|---|---|
| `--yes`, `-y` | Accept defaults, no prompts. |
| `--all-agents` | Install for every detected agent. |
| `--target <name>` | Install for a specific agent. Repeatable. |
| `--skip-cli` | Do not install the CLI. |
| `--skip-agents` | Do not install the skill into any agent. |
| `--skip-login` | Do not prompt to sign in. |
| `--api-key <key>` | Save a `chf_` key during setup. |

## Keep it up to date

To pull the latest CLI and re-render the skill into every detected agent:

```bash
npx @marshell/chifu-wizard@latest update
```

## Let any agent set itself up

If your agent is not in the list, it can still onboard itself. This prints a
self-contained prompt you can paste into any coding agent, which then installs
and verifies chifu on its own:

```bash
npx @marshell/chifu-wizard@latest --agent
```

The command has no side effects of its own. It only prints the prompt.

## What counts as a scan

One pipeline run. Manual or scheduled, succeeded or failed, dashboard
or API key — all consume one slot. The monthly counter resets on the
first of each month (UTC).

When you run out:

- Manual scans return `monthly scan quota exceeded`.
- Scheduled blocks stay on the calendar and resume next month.

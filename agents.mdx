## Default: skills.sh

By default the wizard installs via [skills.sh](https://www.skills.sh) — one shot into all detected agents (Claude Code, Cursor, Codex, Copilot, Windsurf, Gemini, Cline, and more).

Fallback (when skills.sh is unavailable, or you pass `--target` / set `CHIFU_NO_SKILLS_SH=1`):

| Agent | Format | Location |
| --- | --- | --- |
| Claude Code | Skill | `~/.claude/skills/chifu-dep-guard/SKILL.md` |
| Cursor | Rule | `~/.cursor/rules/chifu-dep-guard.mdc` |
| Windsurf | Memory | `~/.codeium/windsurf/memories/chifu-dep-guard.md` |
| Codex | `AGENTS.md` block | `~/.codex/AGENTS.md` |
| OpenCode | `AGENTS.md` block | `~/.config/opencode/AGENTS.md` |
| Gemini CLI | `GEMINI.md` block | `~/.gemini/GEMINI.md` |
| Cline | Rule file | `~/.clinerules/chifu-dep-guard.md` |

`AGENTS.md` / `GEMINI.md` writes are idempotent — delimited `<!-- chifu:begin -->` … `<!-- chifu:end -->` blocks.

## Install for specific agents

```bash
npx @marshell/chifu-wizard@latest --yes --all-agents
npx @marshell/chifu-wizard@latest --target claude --target cursor
```

Valid targets: `claude`, `cursor`, `windsurf`, `codex`, `opencode`, `gemini`, `cline`.

| Flag | Effect |
| --- | --- |
| `--yes`, `-y` | No prompts |
| `--all-agents` | Every detected agent |
| `--target <name>` | Specific agent(s) |
| `--skip-cli` | Skill only, no CLI |
| `--skip-agents` | CLI only, no skill |
| `--api-key <key>` | Save `chf_` key during setup |

## Update

```bash
npx @marshell/chifu-wizard@latest update
```

## Onboard any agent

Print a self-contained prompt (no side effects):

```bash
npx @marshell/chifu-wizard@latest --agent
```

Paste into any coding agent to have it install and verify chifu itself.
